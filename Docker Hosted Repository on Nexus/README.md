# Create Docker Hosted Repository on Nexus


# 📖 Summary

Worked on **Create Docker Hosted Repository on Nexus**, focusing on **Nexus Repository Manager, Docker registry hosting, user role management, and secure Docker image distribution**.
Tasks included configuring a Docker hosted repository in Nexus, setting up user roles and authentication, enabling secure connectors and token-based authentication, and pushing Docker images to the private registry.
Set up Nexus as a secure, local Docker registry and configured Docker daemon to trust the insecure registry, aiming to centralize image management and support secure, controlled access for development workflows.

---

# 🛠️ Tools Used

* **Nexus Repository Manager**: ✨ *Configured Docker hosted repositories, roles, users, and security realms.*
* **Docker CLI**: ⚙️ *Built, tagged, authenticated, and pushed Docker images to the Nexus private registry.*
* **System Networking Tools**: 🚀 *Used netstat and daemon config edits to verify and enable registry connectivity.*

---

# 🎯 Skills Gained

* **Repository Management in Nexus**: 💡 *Learned how to create hosted Docker registries and configure repository connectors.*
* **Role-Based Access Control (RBAC)**: 🔍 *Developed custom roles and users to securely manage Docker repository access.*
* **Docker Authentication & Daemon Configuration**: 🚀 *Configured Docker to trust insecure registries and authenticate using bearer tokens.*

---

# ⚠️ Challenges Faced

* **Repository Validation Errors**: 🛠️ *Resolved configuration errors by properly filling required fields in Nexus UI.*
* **Docker Insecure Registry Setup**: 🔄 *Edited daemon configs and restarted services to allow pushing/pulling over HTTP.*

This lab deepens knowledge in **DevOps and containerized infrastructure management**, showing how Nexus Repository Manager acts as a secure, local Docker registry.
Mastering secure repository setup, user access controls, and Docker client configuration enables efficient image management and secure deployments critical for enterprise-grade CI/CD pipelines. 🚀🔐🐳
